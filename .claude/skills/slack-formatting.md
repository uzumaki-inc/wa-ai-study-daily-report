# スキル: Slack投稿フォーマット

Slack投稿のフォーマットに関するスキルです。

## 投稿構成

5カテゴリ × 各2件をSlack Block Kitで投稿する。

| カテゴリ | Emoji | ソース |
|---|---|---|
| Xで話題 | 🔥 | 英語・日本語 |
| Anthropic | 🟠 | 英語・日本語 |
| モデル・技術 | 🧠 | 英語・日本語 |
| ブログ記事 | 📝 | **日本語のみ** |
| その他話題のニュース | 💬 | 英語・日本語 |

## フォーマットルール

- 記事タイトルは日本語に翻訳する（英語のままにしない）
- 要約は2〜3文、技術者が価値を感じる内容に絞る
- 英語記事には `🇯🇵 日本語で読む` リンクを付与する
- 各カテゴリ末尾に `＋ もっと見る（10件）` リンクを付与する
- リンク先はGitHub Pagesの静的HTMLページ

## 翻訳リンク生成

```
https://translate.google.com/translate?sl=en&tl=ja&u={元のURL}
```

- `lang === 'en'` → 翻訳リンクあり
- `lang === 'ja'` → 翻訳リンクなし
